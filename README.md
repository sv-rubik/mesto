# Сервис "Mesto"

## Описание:
* Интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

## Технологии:
1. HTML
2. CSS / Grid / Flexbox
3. БЭМ методология (включая файловую структуру Nested BEM)
4. Кроссбраузерная адаптивная верстка
5. Javascript
